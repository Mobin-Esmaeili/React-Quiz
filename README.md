React Quiz App

An interactive quiz application built with React and JSON Server. Users can answer multiple-choice questions, get instant feedback on correctness, and move to the next question. The app uses useReducer for state management, ensuring a clean and predictable flow.

🚀 Features

Fetch quiz questions dynamically from a JSON Server API

Answer multiple-choice questions

Immediate feedback (Correct / Wrong)

Move to next question after answering

Track quiz progress with reducer-based state management
